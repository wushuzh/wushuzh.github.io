Algorithm

leetcode #136 Single Number 要求找出给定数组中落单的元素。

def single_number_hashtbl(nums: List[int]) -> int:
    hash_table = {}
    for i in nums:
        try:
            hash_table.pop(i)
        except:
            hash_table[i] = 1
        
    return hash_table.popitem()[0]

Review

Tip

systemctl 已被主流的 linux 系统采用作为系统服务管理工具。

  • 查看服务的配置文件，可以使用 cat 子命令；
  • 设定服务直接启动，同时在下次启动时自动启动一般使用 enable 和 start ，但实际可以通过 enable --now 一次性完成；
  • 如果服务支持配置热加载，而不用重启，可以使用 reload 子命令；
  • 如果查看某个服务依赖的其他服务，可以使用 list-dependencies 子命令；

详见:

  • Justin Ellingwood (2015-02-01) How To Use Systemctl to Manage Systemd Services and Units
  • Linode Introduction to systemctl
  • tutorialspoint 提供了网页版的 linux shell 环境可以做一些简单命令的操作试验，但对于 systemctl 这类指令都执行不了。
