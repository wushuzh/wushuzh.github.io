Algorithm

leetcode #771 jewels and stones 给定两个字符串：

  • J 每个字母代表一种宝石；
  • S 每个字母代表一块原石；

如果某个原石字母是 J 中一员，说明这块原石是宝石，计算 S 中所有宝石的数量。

def nums_jewels_in_stones(J: str, S: str) -> int:
    cnt = 0
    for i in S:
        if i in J:
            cnt += 1
    return cnt

Review

Tip

给定一个列表，其中的元素可能是一个数字，字符，字节，还可能是一个子列表，其中同样包含了数字，字符和子列表，要求用 python 实现一个生成器，将这个列表扁平化。

>>> def flatten(items):
...     for i in items:
...         if isinstance(i, collections.Iterable) and not isinstance(i, (str, bytes)):
...             yield from flatten(i)
...         else:
...             yield i
...
>>> list(flatten([1, 2, [3, 4], "five", [b'six', [7, 8]]]))
[1, 2, 3, 4, 'five', b'six', 7, 8]
>>>
