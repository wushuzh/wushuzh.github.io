插入图片

人是视觉动物。

“有图有真相”——比如新闻网页必然含有相关照片或视频截图。

另外就是在网站发展初期，只要是不涉及国家领导人的新闻，其网页的边栏、底栏都会有一些清凉图片(有时配合耸人听闻的标题打包显示)，感觉多数新闻聚合类网站都会这招，我猜这些小视频和缩略图对网站的流量有不小的贡献。

“一图胜千言”——图片在解释某些复杂事物的时候优势非常明显。 比如 HipHop 式握手

校对

无论 Hugo 、 Jekyll 或其他任何静态站点工具无一例外都提供 serve 命令来在本机启动一个轻量级的 Web 服务器，这样你就能在提交前，方便地预览一下最终展示效果。当你对内容、配置，主题等做微调的时候，被修改的内容也是即刻在本地服务器中生效( 不用重启 )，非常方便。

写作只需要打开你趁手的任何编辑器，如 Atom , neovim 甚至记事本。Huge 的所有博文都保存在 content 子目录，Jekyll 则设定了草稿和正式发布两个子目录。另外 Huge 提供模板 archetypes 用于设定不同类型的博文自定义的文章模板。
